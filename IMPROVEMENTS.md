## Color Palette

**Primary:**
- Orange-600 (#EA580C)
- Red-600 (#DC2626)

**Neutrals:**
- Slate-900 (#0F172A) - Dark backgrounds
- Slate-700 (#334155) - Secondary text
- Slate-600 (#475569) - Body text
- Slate-200 (#E2E8F0) - Borders
- Slate-50 (#F8FAFC) - Light backgrounds
- White (#FFFFFF) - Cards, highlights

**Accents:**
- Blue-600, Purple-600, Green-600 (for service categories)

## Typography Scale

**Display (Headings):**
- H1: text-5xl lg:text-7xl (48px-72px)
- H2: text-4xl lg:text-6xl (36px-60px)
- H3: text-2xl (24px)

**Body:**
- Large: text-xl (20px)
- Regular: text-base (16px)
- Small: text-sm (14px)

## Spacing System

**Section Padding:** py-32 (128px)
**Card Padding:** p-8 (32px)
**Element Gaps:** gap-8 (32px) for grids
**Margins:** mb-20 (80px) for section breaks

## Professional Features

1. **Micro-interactions**: Subtle scale on hover (1.02)
2. **Smooth transitions**: 300ms duration
3. **Shadows**: Layered shadows with color (shadow-orange-500/25)
4. **Glassmorphism**: backdrop-blur-xl with bg-white/95
5. **Grid patterns**: Subtle background patterns
6. **Gradient orbs**: Soft, slow-moving background elements
7. **Professional icons**: Lucide React icons throughout
8. **Animations**: Framer Motion for scroll-triggered effects

## Performance Optimizations

- Minimal CSS with Tailwind
- Next.js 15 with Turbopack for fast dev
